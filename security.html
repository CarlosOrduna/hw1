Computer Security

Computer Security focuses on countermeasures based on actions, products, devices or techniques that reduce threats, vulnerabilities or attacks by lowering the risks, discovering and reporting the vulnerabilities and eliminating the threats imposed.
This can be achieved via secure coding, keeping software up to date, training users, and keeping a good response protocol to any attack.
